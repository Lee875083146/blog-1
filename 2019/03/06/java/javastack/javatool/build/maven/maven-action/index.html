1. 常见问题

1.1. IDEA 修改 JDK 版本后编译报错

错误现象

修改 JDK 版本，指定 maven-compiler-plugin 的 source 和 target 为 1.8 。

然后，在 Intellij IDEA 中执行 maven 指令，报错：

错误原因

maven 的 JDK 源与指定的 JDK 编译版本不符。
